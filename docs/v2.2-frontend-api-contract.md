## 1. 鉴权模块 — 🆕 全新

### 1.1 `POST /api/auth/login`

**Phase**: 0 | **优先级**: P0（全部页面前置）

```
请求体:
  飞书SSO:  { provider: "feishu", code: string }      // OAuth callback code
  独立账号: { provider: "password", username: string, password: string }

响应:
{
  token: string,         // JWT, 有效期 24h
  user: {
    id: string,
    name: string,
    role: "admin" | "qc_inspector" | "middle_platform" | "partner" | "developer",
    avatar_url?: string,
    permissions: string[]   // 可访问路由列表
  }
}
```

**后端实现要点**：
- 飞书 OAuth2：调用飞书开放平台 `authen/v1/access_token` 获取用户身份
- 合作方账号：新建 `users` 表或复用现有 PG 表，bcrypt 存储密码
- JWT 签发：HS256，payload 含 `{user_id, role, exp}`
